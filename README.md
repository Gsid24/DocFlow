# DocFlow — Система Электронного Документооборота (СЭДО)

Внутренняя система для загрузки, хранения и согласования документов с использованием ЭЦП.

## Статус проекта (17.06.2026)

### Что сделано:
- Создан Django проект
- Настроено виртуальное окружение (`venv`)
- Создано два приложения: `documents` и `approvals`
- Реализована базовая модель `Document` (название, файл, автор, дата и т.д.)
- Настроена админка для работы с документами
- Реализован список документов на главной странице
- Настроена загрузка и скачивание файлов (MEDIA)
- Добавлена базовая транслитерация русских имён файлов

### На чём остановились:
Проблема с сохранением оригинального имени файла при загрузке документов с кириллицей (файл сохраняется как `docx`, без названия). Нужно доработать функцию `transliterate_filename`.
