# DocFlow — Система Электронного Документооборота (СЭДО)

Внутренняя система для загрузки, хранения и согласования документов с использованием **ЭЦП (КриптоПро)**.

## Цель проекта
Создать удобный, лёгкий и безопасный инструмент для небольшого коллектива (10–50 человек) с полным циклом работы с документами: загрузка → запуск согласования → подписание КЭП → хранение.

---

### Что уже сделано:
- Создан Django-проект `docflow`
- Настроено виртуальное окружение (`venv`)

  - **`approvals`** — будет отвечать за процессы согласования (задачи на подпись, статусы, история подписей ЭЦП).
